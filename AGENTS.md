# AGENTS.md

## 言語
- チャット、ログ、メッセージ、コメントは日本語で記述する。

## ドキュメント配置
- 仕様書: `docs/specs/`
- ADR: `docs/adr/`
- 引き継ぎ情報: `docs/handoff/`
- 作業計画: `docs/plans/`
- その他の関連ドキュメント: `docs/`

## git commit 前の確認
- 仕様書を作成または更新する。
- 技術選定、アーキテクチャ判断、試行錯誤の経緯など、将来の意思決定に影響する決定をした場合は ADR を作成または更新する。
- 引き継ぎ情報を作成または更新する。内容が大きくなる場合は、作業計画を `docs/plans/` に分けて管理する。
- 関連するドキュメントを更新し、各ドキュメントに鮮度情報（最終確認日・対象バージョンなど）を記載する。

## 作業時の注意
- エラーを解消できない場合は、無理に進めず状況を報告する。
- 大きな変更または削除を行う前に確認を取る。
